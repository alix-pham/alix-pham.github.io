How I built this website

This site is built with Hugo and hosted on GitHub Pages. I initially set it up in 2023, and now maintain and update it with the help of AI (Claude Code).

Setup

Install dependencies

  • Git
  • Hugo

Set up a GitHub Pages repo

  • Follow these instructions

Create the Hugo site and add a theme

  • Run hugo new site <<your repo name>> --force
  • Download a theme, e.g. Poison
    1. If you won’t modify the theme: git clone <<theme-url>> /themes/<<theme name>>
    2. If you will make changes: fork the theme repo, then git submodule add <<forked-theme-url>> themes/<<theme-name>>

Add content and test locally

  • Edit config.toml and .md files in the site directory
  • Run hugo server to preview at http://localhost:1313/
  • Run hugo to build

Modify the theme (requires option 2 above)

cd themes/<<your theme>>
# make your changes
git add .
git commit -m "<<message>>"
git push origin master

If you hit a branch error:

git stash
git checkout master
git stash pop
